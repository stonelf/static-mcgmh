世界记忆冠军：记得做个纯素食者！

　　　

　　
世界记忆冠军 约翰·内斯：记得做个纯素食者!
　　23岁，来自瑞典的约翰·内斯(JonasVonEssen)刚刚成为了世界记忆冠军(World Memory Champion)。他以在不到一分钟时间里记下一整副扑克牌、准确回忆出数以千计的数字等成绩创下了该比赛史上的最高分数。在每次吃东西时，他也不忘想起动物——是的，约翰是一个纯素食者。这证明了以植物为基础的饮食不仅仅是最富有同情心，也是最聪明的!
　　约翰就他富有同情心的素食饮食回答了以下问题：
　　你为什么决定做一名纯素食者?
　　很简单，因为我不想造成其他生物被折磨和杀戮。不幸的是，我呆滞的脑袋在这个星球上活了20年才足够理智地决定要做一名蛋奶素食者，又花了一年半才明白如果不想造成动物被折磨，纯素食才是应该走的路。
　　有没有一些书、乐队、名人或者运动员影响或者激励你成为纯素食者?
　　StephDavis——美国登山运动员、滑翔服飞行员，也是一名纯素食者——对我是个启发。在我从一个蛋奶素食者变为纯素食者时，我读了她的一本名为《学会飞翔》的书(那正好也是我开始高空跳伞的时候)。她并不是导致我作出这个决定的人，但因为她，让我感到这个决定更容易做出了。
　　你最喜欢哪些食物?
　　我非常喜欢亚洲食物，比如有很多香料的炒豆腐。我也很喜欢制作代替奶酪的东西，比如用坚果和香草制作意式素奶酪千层面和各种意大利面。
　　你开始吃纯素食后感受到了什么益处?
　　最大的好处大概是我感觉到我在做一些让世界更美好的事(或者说停止做让世界不美好的事)。回顾我记忆训练的结果，我只能说自从我吃纯素食以来，我在七个世界级比赛中赢了六个，其中最近一个就是我在世界记忆冠军大赛得到史上最高分数(也就意味着比所有其他食肉的人都高!)。
　　你对想要帮助动物却不知道从何做起的人有什么建议?
　　开始纯素食!我认为要让事情变好，最简单的方法就是纯素食了。你不仅是在防止伤害动物，而且在最小化你对环境的影响。它比你想象的容易太多!我感觉纯素食完全不是一种牺牲，因为我现在吃的东西比以前还吃肉时更好吃。谁知道呢——纯素食可能还会让你拥有全世界上最好的记忆力!
