美国菩萨寺 以佛学书籍帮助受刑人 106位求皈依受

　

菩萨寺住持慧光法师、美国菩萨寺执行长显中法师以及居士们于布道的监狱外面合影

法师整理居士们捐赠的英文佛教书籍

　　菩萨寺的美国圣地亚哥分寺专门在监狱里面成立佛教书籍的图书馆帮助受刑犯能够洗心革面。

　　菩萨寺住持慧光法师出生于中国台湾，1994年于美国加州大学毕业后，礼星云大师座下披剃出家，于佛光山丛林学院进修毕业。2008年得法自中国大陆深圳弘法寺本焕长老，为临济宗四十五代禅人。

　　美国菩萨寺的显宗法师于2007年开始担任美国菩萨寺执行长一职，并致力于监狱弘法之法务约5年多的时间。显中法师说监狱的受刑人中，刚开始时可以说没有一个是真正的佛教徒。他们有的是基督教徒、天主教徒、回教徒，甚至有一些并没有宗教信仰。我们在监狱弘法，刚开始并不强调要相信佛教，而是告诉他们佛法的究竟真理。佛陀的教法是让我们去认识并了解生命的真相；如何行善止恶；如何与其他众生互动。“刚开始我们不会强调‘这是佛陀说的'或是‘这是佛经上说的'；而是用一种比较简单的方式，将佛法带入他们的生活。譬如我跟他们说：你可以不用相信佛陀，因为你不曾遇见佛陀。但是你身旁的每个人，你骂他，他会生气，他骂你，你也会生气，所以不可以随便骂人。我教导他们因缘果报的道理； 教导他们如何做一个好人。我们从没有给他们信仰的压力，而是跟他们分享佛法，帮助他们如何在生活当中能够少烦少恼；如何让自己与别人都能得到平安跟快乐。这跟其他宗教完全强调信者得救的方式全然不同，当他们正在迷惘中寻找生命的方向时，遇到这种教法，比较能够接受，然后愿意开始学习。”

　　另外，法师也经常劝勉受刑人作“转念”的学习，“把监狱转成修行 道场”。“其实美国监狱的设施还算不错。不管是吃的、住的或是衣服都蛮干净的。去年佛诞节时，我曾跟他们一起吃饭，看到他们吃的食物非常丰富。我常常告诉他们，其实在监狱的作息从早到晚跟寺庙丛林并没有太大差异，所以只要将观念改变一下就行了。监狱作息是集体的，譬如早上六点钟起床，六点半吃早饭，大家都穿同样的制服，吃同样的饭菜，这些作息跟寺庙丛林的规矩大约相同。而且他们的囚室(cell) 很小，我觉得他们就好像在关房里面一样。我总是鼓励他们，你们要改变观念，要感谢一切的因缘条件，感谢有得吃、有得住、有衣服穿、有水有电。若能接受‘这里就是我的修行道场’，那么他们就跟我没有什么不 同了。他们很感谢我这样地分享，因为他们从来没有这样想过，也从来没有遇到一位出家众分享他出家的生活。这是一个观念问题，如果他们能转这个观念，对他们很有帮助。并且显中法师在印度学习过一些内观(Vipassana)方法教导大众练习身心的观照。面对面跟我们互动的，四个监狱加起来大约有五百多位。写信来 互动的就更多了，总共一千多位。监狱的受刑人因为受过苦，所以比较容易接受佛法的熏习。佛陀初转法轮就是讲四圣谛“苦、集、灭、道” 。苦，是我们最好的老师，认识苦就会想离开苦，就会不断地寻找方法，而佛法对苦的原因及如何息灭苦的方法都讲得非常清楚、非常有次第。所以他们有机会接触到佛法，就能信受奉行。他们之中有的很用功，清晨两三点就起来做功课了。

　　经过了每个月的佛法课程和佛经的研读之后，慢慢地这些受刑人开始愿意皈依三宝，成为真正的佛教徒。从2008年开始，每年都有二三十位受刑人皈依慧光法师。而慧光法师每年的五月都会特别从中国台湾去美国为他们传戒。因为有了图书馆的设立，因缘更加殊胜，三所监狱加起来总共有106 人请求皈依并受五戒。

　　显中法师提到一所监狱通常有A、B、C、D 四处牢房， 所以要设立十六个佛学图书馆。但是图书进入监狱需要通过一些审查过程。目前已设置好的有 Calipatria State Prison 及Chuckawalla Valley State Prison 各有四个图书馆，加上 Centinela State Prison 有一个，总共加起来有九个图书馆。他们表示，正在等待审查，若能通过则其他七个图书馆也可以设立起来。

　　对于书籍的部分，显中法师很感谢很多团体及很多人的发心，很多人不断从各地寄书来支持这个捐书活动。除了美国地区之外，还有从加拿大、新加坡、印度尼西亚、马来西亚，及中国香港、台湾等国家和地区寄来的英文佛学书籍。显中法师也希望更多的大众能够发心捐赠英文佛教书籍给监狱图书馆。
